SmartEye - Software para monitoramento das condições ergonômicas individuais

Sistema web que realiza os cálculos dos dados do conforto térmico em tempo real e de maneira individualizada para os trabalhadores de uma instituição. Esse sistema apresenta uma inovação a cerca do que é disponível na NR-17, bem como da maneira com a qual são realizadas as análises de conforto térmico atualmente. De acordo com o disposto na norma, essa análise é realizada de maneira geral, isto é, sem considerar as particularidades de cada atividade laboral, nem as características dos trabalhadores. Além disso, essas análises são realizadas/revisadas com baixíssimas frequências, sendo um dos fatores que ocasiona essa baixa periodicidade é o custo para realização dos mesmos.
